# Student Record Management System

A console-based Student Record Management System built in C++.

## Features
- Add Student Record
- Display Student Records
- Search Student by School ID
- Update Student Details
- Calculate Class Average
- Generate Rank List

## Concepts Used
- Structures
- Functions
- Arrays
- Selection Sort
- Input Validation
- String Handling
- iomanip Formatting

## Future Improvements
- Delete Record
- File Saving (fstream)
- Vector-Based Storage
- OOP Version
